Remarques :
Dans la conception de ce site, plusieurs principes d'écoconception ont été suivis, notamment :

Utilisation d'un code simple et optimisé pour limiter la consommation de ressources.
Mise en place d'un design minimaliste avec des couleurs et images légères pour réduire la consommation énergétique lors du chargement.
Réutilisation de structures HTML et CSS communes entre toutes les pages pour limiter la duplication de code.
Adoption d'une mise en page claire pour une meilleure lisibilité et accessibilité.
